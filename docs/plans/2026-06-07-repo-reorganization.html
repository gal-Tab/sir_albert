T4 Create kw-compound skill skills/knowledge/kw-compound/SKILL.md (new)
✨ New skill — adapted from EveryInc/compound-knowledge-plugin's kw:compound for sir_albert's raw/ → wiki/ pipeline
What kw-compound does

Closes the knowledge loop. After a session produces insights, /kw-compound captures 1–3 items, classifies them as concept | entity | comparison | source, validates with the user, and writes them to raw/<slug>.md using wiki-schema.md's frontmatter. Next /kb-compile run ingests them into wiki/.

Key guardrails
  • Writes to raw/ only — never directly to wiki/
  • User approval required before every save
  • Uses STATED / INFERRED / UNCERTAIN confidence labels
  • Max 3 items per session
  • SKILL.md created at skills/knowledge/kw-compound/
  • Adapts compound-knowledge pattern to sir_albert's wiki-schema.md

T6 Create tools/publish-skills.sh tools/publish-skills.sh (new, executable)
Usage
tools/publish-skills.sh --all            # publish all skills
tools/publish-skills.sh html-plans       # publish one skill
What it does

Finds each skill by name under the nested skills/<category>/ structure and copies it to ~/.claude/skills/<name>/ (flat). Claude Code reads from the flat directory — the category hierarchy is for repo organization only.

  • Script created and chmod +x
  • Handles both --all and single-skill modes
